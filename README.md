# 🏥 Smart Health Support System

A mini healthcare web application that automates request handling by analyzing user input and routing it to the appropriate department with priority and response generation.

Link:- https://healthcare-kappa-sage.vercel.app

## 🚀 Features

- 📩 Submit health/support requests
- 🤖 Automated request analysis (AI-like logic)
- 🧠 Smart routing to departments (Medical / Food / Volunteer)
- ⚡ Priority detection (High / Medium / Low)
- 🏢 Department panel to manage requests
- 🔄 Status updates (Pending → In Progress → Completed)
- 🔍 Track request using unique Request ID
- 💾 File-based storage (JSON)

---

## 🧠 AI / Automation Feature

The system analyzes user input using keyword-based logic and:

- Classifies request type
- Assigns priority level
- Routes to appropriate department
- Generates response automatically
